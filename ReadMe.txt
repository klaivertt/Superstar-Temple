Projet "Superstar Temple" fait par Alexandre Bosio, Alex Mathieu, Mattéo Descieux, Guillaume Benito Garcia et Hugo Rovira


DESCRIPTION :

C'est un escape game dans le quel la coopération est la clé. Ce jeu se joue en split screen, en coopération.


TOUCHES :

ZQSD => mouvements (joueur 1)
E => prendre/lâcher des objets (joueur 1)
flèche haut/bas/gauche/droite => mouvements (joueur 2)
NumPad 0 => prendre/lâcher des objets (joueur 2)
